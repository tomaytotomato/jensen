# Jensen Theme for Slack

Bring the Cyber Renaissance to Slack with this theme inspired by the iconic aesthetics of **Deus Ex: Human Revolution**. Featuring signature gold and copper tones set against dark backgrounds, Jensen transforms your workspace into an augmented experience.

![Jensen Theme](../public/jensen_large.png)

**Creator:** tomaytotomato

## Features

- 🎨 **Cyber Renaissance Aesthetic**: Rich gold (`#EDA622`) and copper tones against deep carbon black backgrounds
- 🌙 **Dark Theme Optimized**: Easy on the eyes during extended work sessions
- 🎯 **Carefully Crafted Colors**: Optimized for Slack's sidebar and messaging interface
- 💻 **Cross-Platform**: Works on Slack for Windows, macOS, Linux, and web

## Installation

1. Open Slack
2. Click your workspace name in the top-left corner
3. Go to **Preferences** (or press `Ctrl+,` / `Cmd+,`)
4. Navigate to **Themes** in the sidebar
5. Scroll down to **Create a custom theme**
6. Copy the color values from `jensen-slack.txt`
7. Paste them into the color input field
8. The theme will be applied instantly!

## Color Values

From `jensen-slack.txt`:

```
#141514,#1C1D1C,#EDA622,#FBFBFB,#E55D02,#FBFBFB,#A0FF33,#ED333C,#141514,#FBFBFB
```

### What Each Color Represents

1. **Column BG**: `#141514` (Eerie Black)
2. **Menu BG Hover**: `#1C1D1C` (Slightly lighter black)
3. **Active Item**: `#EDA622` (Signature Marigold)
4. **Active Item Text**: `#FBFBFB` (White)
5. **Hover Item**: `#E55D02` (Electronic Copper)
6. **Text Color**: `#FBFBFB` (White)
7. **Active Presence**: `#A0FF33` (French Lime)
8. **Mention Badge**: `#ED333C` (Imperial Red)
9. **Top Nav BG**: `#141514` (Eerie Black)
10. **Top Nav Text**: `#FBFBFB` (White)

## Reverting to Default

To revert to Slack's default theme:
1. Go to **Preferences** → **Themes**
2. Select one of Slack's built-in themes
3. Or click **Reset to default**
