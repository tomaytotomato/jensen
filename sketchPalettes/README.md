# Jensen Theme for Sketch Palettes

Bring the Cyber Renaissance to Sketch with this color palette inspired by the iconic aesthetics of **Deus Ex: Human Revolution**. Featuring signature gold and copper tones, Jensen provides the perfect color scheme for your UI/UX design projects.

![Jensen Theme](../public/jensen_large.png)

**Creator:** tomaytotomato

## Features

- 🎨 **Cyber Renaissance Aesthetic**: Rich gold (`#EDA622`) and copper tones against deep carbon black backgrounds
- 🎯 **Complete Palette**: All 11 official Jensen colors
- 💎 **Design Ready**: Perfect for UI/UX, web design, and digital art
- 💻 **Sketch Compatible**: Works with Sketch app on macOS

## Installation

### Using Sketch Palettes Plugin

1. Install the [Sketch Palettes plugin](https://github.com/andrewfiorillo/sketch-palettes) if you haven't already

2. In Sketch, go to **Plugins** → **Sketch Palettes** → **Load Palette**

3. Select `jensen.sketchpalette` from this directory

4. The Jensen colors will be loaded into your document

### Manual Installation

1. Copy `jensen.sketchpalette` to your desktop or preferred location

2. In Sketch, open the color picker

3. Click the gear icon in the Global Colors section

4. Select **Import Palette from File**

5. Choose `jensen.sketchpalette`

## Design Tips

- **Primary Color**: Use Marigold (`#EDA622`) as your main brand color
- **Dark Mode**: Eerie Black (`#141514`) makes an excellent dark mode background
- **Contrast**: White text on Eerie Black provides excellent readability
- **Accents**: Use French Lime for success, Imperial Red for errors
- **Hierarchy**: Copper and Dark Goldenrod work great for secondary elements
