# Jensen Theme for iTerm2

Bring the Cyber Renaissance to iTerm2 with this theme inspired by the iconic aesthetics of **Deus Ex: Human Revolution**. Featuring signature gold and copper tones set against dark backgrounds, Jensen transforms your terminal into an augmented experience.

![Jensen Theme](../public/jensen_large.png)

**Creator:** tomaytotomato

## Features

- 🎨 **Cyber Renaissance Aesthetic**: Rich gold (`#EDA622`) and copper tones against deep carbon black backgrounds
- 🌙 **Dark Theme Optimized**: Easy on the eyes during extended terminal sessions
- 🎯 **Carefully Crafted Colors**: Optimized for readability and visual hierarchy
- 💻 **macOS Compatible**: Works with iTerm2 on macOS

## Installation

1. Download `jensen.itermcolors` from this directory
2. Open iTerm2
3. Go to **iTerm2** → **Preferences** → **Profiles** → **Colors**
4. Click **Color Presets...** dropdown (bottom-right)
5. Select **Import...**
6. Navigate to and select the `jensen.itermcolors` file
7. Select **Jensen** from the Color Presets dropdown
8. Close Preferences to apply

The Jensen theme uses a carefully curated palette inspired by Deus Ex: Human Revolution's iconic UI design:

|-------------------|-------------|----------------------------------------------------------|
| **Eerie Black**   | `#141514`   | Main background, canvas                                  |

| **Pacific Blue**  | `#0EA2C9`   | Information states, links, functions                     |
| **Imperial Red**  | `#ED333C`   | Error states, warnings, critical alerts                  |

## Tips

- **Font Recommendation**: For the full Deus Ex experience, use a monospace font like Menlo, Monaco, or Fira Code
- **Transparency**: iTerm2's transparency works great with Jensen - try 10-20% for a sleek look
- **Profiles**: You can set Jensen as your default profile or create multiple profiles with different fonts
