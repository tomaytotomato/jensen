# Jensen - iTerm Color Theme

Bring your terminal into the Cyber Renaissance with this color theme inspired by the iconic aesthetics of Deus Ex: Human Revolution. Featuring rich gold and copper tones set against dark backgrounds, Jensen transforms your iTerm into an augmented experience.

**Creator:** tomaytotomato

## Installation

1. Launch iTerm
2. Press `command`-`I` to open the iTerm preferences
3. Navigate to **Profiles** → **Colors** tab
4. Click on **Color Presets...** dropdown → **Import...**
5. Select the `jensen.itermcolors` file from this directory
6. Click on **Color Presets...** again and select **Jensen** from the list

## Color Palette

The Jensen theme uses the following carefully selected colors inspired by Deus Ex: Human Revolution:

- **Background:** `#141514` - Deep carbon black
- **Foreground:** `#FBFBFB` - Bright white
- **Yellow/Gold:** `#EDA622` - Signature gold accent
- **Green:** `#A0FF33` - French lime
- **Blue:** `#0EA2C9` - Electric blue
- **Red:** `#F25244` - Alert red
