# Jensen Theme for Xcode

Bring the Cyber Renaissance to Xcode with this theme inspired by the iconic aesthetics of **Deus Ex: Human Revolution**. Featuring signature gold and copper tones set against dark backgrounds, Jensen transforms your iOS/macOS development environment into an augmented experience.

![Jensen Theme](../public/jensen_large.png)

**Creator:** tomaytotomato

## Features

- 🎨 **Cyber Renaissance Aesthetic**: Rich gold (`#EDA622`) and copper tones against deep carbon black backgrounds
- 🌙 **Dark Theme Optimized**: Easy on the eyes during extended coding sessions
- 🎯 **Carefully Crafted Colors**: Optimized for Swift, Objective-C, and other languages
- 💻 **macOS Native**: Designed specifically for Xcode on macOS

## Installation

1. Close Xcode if it's running

2. Copy the theme file(s) to Xcode's themes directory:
   ```bash
   mkdir -p ~/Library/Developer/Xcode/UserData/FontAndColorThemes/
   cp *.xccolortheme ~/Library/Developer/Xcode/UserData/FontAndColorThemes/
   ```

3. Launch Xcode

4. Open **Xcode** → **Preferences** (or press `Cmd+,`)

5. Go to **Themes** tab

6. Select **Jensen** from the list

7. Close Preferences - the theme is now applied!

## Alternative Installation

You can also install by double-clicking the `.xccolortheme` file, and Xcode will automatically import it.

## Tips

- **Font Recommendation**: Use SF Mono (built into macOS) or Menlo for the best native experience
- **Font Size**: Adjust in Preferences → Fonts & Colors
- **Line Spacing**: Increase line spacing slightly for improved readability
- **Dark Mode**: Enable macOS Dark Mode for a fully immersive augmented experience
- **Minimap**: Jensen works beautifully with Xcode's minimap feature

## Customization

To customize the theme:

1. Export Jensen from Xcode Preferences → Themes
2. Edit the `.xccolortheme` file (it's an XML property list)
3. Re-import to Xcode
4. Or create a new theme based on Jensen
