# Jensen Theme for Warp

Bring the Cyber Renaissance to Warp with this theme inspired by the iconic aesthetics of **Deus Ex: Human Revolution**. Featuring signature gold and copper tones set against dark backgrounds, Jensen transforms your terminal into an augmented experience.

![Jensen Theme](../public/jensen_large.png)

**Creator:** tomaytotomato

## Features

- 🎨 **Cyber Renaissance Aesthetic**: Rich gold (`#EDA622`) and copper tones against deep carbon black backgrounds
- 🌙 **Dark Theme Optimized**: Easy on the eyes during extended terminal sessions
- 🎯 **Carefully Crafted Colors**: Optimized for readability and visual hierarchy
- 💻 **macOS & Linux Compatible**: Works with Warp terminal on macOS and Linux

## Installation

### Method 1: Manual Installation (Recommended)

1. Copy `jensen-warp.yaml` to Warp's themes directory:
   - **macOS**: `~/.warp/themes/`
   - **Linux**: `~/.local/share/warp-terminal/themes/`
2. Create the directory if it doesn't exist:
   ```bash
   mkdir -p ~/.warp/themes
   cp jensen-warp.yaml ~/.warp/themes/
   ```
3. Open Warp
4. Press `Cmd+P` (macOS) or `Ctrl+P` (Linux) to open the command palette
5. Type "Open Settings" and press Enter
6. Go to **Appearance** → **Themes**
7. Search for **Jensen** and select it

### Method 2: Direct Copy

1. Open Warp Settings (`Cmd+,` on macOS)
2. Navigate to **Appearance** → **Themes**
3. Click **Create New Theme**
4. Copy the contents of `jensen-warp.yaml` into the editor
5. Save as "Jensen"
