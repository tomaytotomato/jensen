# Jensen Theme for Windows CMD

Bring the Cyber Renaissance to Windows Command Prompt with this theme inspired by the iconic aesthetics of **Deus Ex: Human Revolution**. Featuring signature gold and copper tones set against dark backgrounds, Jensen transforms your command prompt into an augmented experience.

![Jensen Theme](../public/jensen_large.png)

**Creator:** tomaytotomato

## Features

- 🎨 **Cyber Renaissance Aesthetic**: Rich gold (`#EDA622`) and copper tones against deep carbon black backgrounds
- 🌙 **Dark Theme Optimized**: Easy on the eyes during extended terminal sessions
- 🎯 **Carefully Crafted Colors**: Optimized for readability in CMD
- 💻 **Windows Native**: Works on Windows 10 and Windows 11

## Installation

### Automatic Installation (Recommended)

1. **Back up your registry** (important!)
2. Double-click `jensen.reg` from this directory
3. Click **Yes** when prompted to add to registry
4. Open a new Command Prompt window
5. The Jensen colors are now applied!

### Manual Installation

If you prefer to apply manually:

1. Open Command Prompt
2. Right-click the title bar → **Properties**
3. Go to the **Colors** tab
4. Manually set each color using the values from `jensen.reg`
5. Click **OK** to apply

## Important Notes

⚠️ **Registry Warning**: Modifying the Windows Registry can affect system stability. Always back up before making changes.

⚠️ **Applies Globally**: This theme applies to all Command Prompt windows system-wide.

⚠️ **Windows Terminal**: For a better experience with more customization options, consider using Windows Terminal with the Jensen theme (available in the `winTerminal` directory).

## Reverting Changes

To revert to default CMD colors:

1. Open Command Prompt
2. Right-click title bar → **Defaults**
3. Go to **Colors** tab
4. Click **Reset** or manually restore original colors
5. Click **OK**

Or restore from your registry backup.

## Tips

- **Font**: Change to a better monospace font in Properties → Font tab
- **Window Size**: Increase buffer size for more scrollback
- **Upgrade**: Consider upgrading to Windows Terminal for modern features
- **PowerShell**: For PowerShell, these colors will also apply to the PowerShell console host
