# Jensen Theme for Konsole

Bring the Cyber Renaissance to Konsole with this theme inspired by the iconic aesthetics of **Deus Ex: Human Revolution**. Featuring signature gold and copper tones set against dark backgrounds, Jensen transforms your KDE terminal into an augmented experience.

![Jensen Theme](../public/jensen_large.png)

**Creator:** tomaytotomato

## Features

- 🎨 **Cyber Renaissance Aesthetic**: Rich gold (`#EDA622`) and copper tones against deep carbon black backgrounds
- 🌙 **Dark Theme Optimized**: Easy on the eyes during extended terminal sessions
- 🎯 **Carefully Crafted Colors**: Optimized for readability and visual hierarchy
- 💻 **KDE Plasma Integration**: Perfect for KDE desktop environments

## Installation

1. Copy `jensen-konsole.colorscheme` to Konsole's color schemes directory:
   ```bash
   mkdir -p ~/.local/share/konsole/
   cp jensen-konsole.colorscheme ~/.local/share/konsole/
   ```
2. Open Konsole
3. Go to **Settings** → **Edit Current Profile**
4. Navigate to the **Appearance** tab
5. Select **Jensen** from the color scheme list
6. Click **Apply** and **OK**

## Setting as Default

To make Jensen your default color scheme:
1. Open Konsole
2. **Settings** → **Manage Profiles**
3. Select your profile
4. Click **Set as Default**
5. Edit the profile and set Jensen as the color scheme

## Color Palette

The Jensen theme uses a carefully curated palette inspired by Deus Ex: Human Revolution's iconic UI design:

| Color Name        | Hex Code    | Usage                                                    |
|-------------------|-------------|----------------------------------------------------------|
| **Eerie Black**   | `#141514`   | Main background, canvas                                  |
| **White**         | `#FBFBFB`   | Primary text, foreground elements                        |
| **Silver Sand**   | `#C4C7C4`   | Comments, muted text, secondary information              |
| **Marigold**      | `#EDA622`   | Primary accent, signature gold highlight, selected items |
| **Electronic Copper** | `#E55D02` | Secondary accent, hover states, warm highlights        |
| **French Lime**   | `#A0FF33`   | Success states, positive feedback, strings               |
| **Pacific Blue**  | `#0EA2C9`   | Information states, links, functions                     |
| **Imperial Red**  | `#ED333C`   | Error states, warnings, critical alerts                  |

## Tips

- **Font Recommendation**: Use Hack, JetBrains Mono, or Fira Code for excellent ligature support
- **Transparency**: Enable transparency in Konsole settings for a sleek augmented effect
- **KDE Integration**: Pair with the Jensen KDE Plasma theme (available in the `kdePlasma` directory)
- **Multiple Profiles**: Create different profiles with Jensen for various tasks
