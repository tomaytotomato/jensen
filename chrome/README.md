# Jensen Theme for Chrome

Bring the Cyber Renaissance to Google Chrome with this theme inspired by the iconic aesthetics of **Deus Ex: Human Revolution**. Featuring signature gold and copper tones set against dark backgrounds, Jensen transforms your browser into an augmented experience.

![Jensen Theme](../public/jensen_large.png)

**Creator:** tomaytotomato

## Features

- 🎨 **Cyber Renaissance Aesthetic**: Rich gold (`#EDA622`) and copper tones against deep carbon black backgrounds
- 🌙 **Dark Theme Optimized**: Easy on the eyes during extended browsing sessions
- 🎯 **Carefully Crafted Colors**: Optimized for Chrome's UI elements
- 💻 **Chrome Compatible**: Works with Google Chrome on Windows, macOS, and Linux

## Installation

### Method 1: Load Unpacked Extension (Development Mode)

1. Download this directory or clone the repository
2. Open Chrome and navigate to `chrome://extensions/`
3. Enable **Developer mode** (toggle in the top-right corner)
4. Click **Load unpacked**
5. Select this `chrome` directory
6. The Jensen theme will be applied automatically

### Method 2: Manual Installation from Files

1. Ensure you have `manifest.json` and theme files in this directory
2. Follow the steps above for loading as an unpacked extension
3. Keep the directory in place - don't delete it after loading

## Tips

- **Chrome Web Store**: This theme may also be available on the Chrome Web Store (check the main repository)
- **Dark Mode**: Pair with Chrome's native dark mode for the best experience
- **Extensions**: Jensen looks great with dark-themed extensions
- **Sync**: Chrome themes sync across devices when signed in

## Troubleshooting

- **Theme Not Applying**: Make sure Developer mode is enabled and the extension is active
- **Updates**: If you update the theme files, reload the extension on `chrome://extensions/`
- **Conflicts**: Disable other themes before applying Jensen
