# Jensen Theme for Alfred

Bring the Cyber Renaissance to Alfred with this theme inspired by the iconic aesthetics of **Deus Ex: Human Revolution**. Featuring signature gold and copper tones set against dark backgrounds, Jensen transforms your macOS launcher into an augmented experience.

![Jensen Theme](../public/jensen_large.png)

**Creator:** tomaytotomato

## Features

- 🎨 **Cyber Renaissance Aesthetic**: Rich gold (`#EDA622`) and copper tones against deep carbon black backgrounds
- 🌙 **Dark Theme Optimized**: Easy on the eyes during quick searches
- 🎯 **Carefully Crafted Colors**: Optimized for Alfred's interface elements
- 💻 **macOS Native**: Designed specifically for Alfred 4+ on macOS

## Installation

Installing the Jensen theme for Alfred is incredibly simple:

1. Download `jensen.alfredappearance` from this directory
2. Double-click the file
3. Alfred will automatically import and apply the theme
4. That's it! You're now augmented.

## Manual Installation (Alternative)

If double-clicking doesn't work:

1. Open Alfred Preferences (`Cmd+,`)
2. Go to the **Appearance** tab
3. Click the **+** button at the bottom
4. Select **Import...** and choose `jensen.alfredappearance`
5. The theme will be imported and applied

## Customization

You can further customize the Jensen theme:

1. Open Alfred Preferences → **Appearance**
2. Select the **Jensen** theme
3. Adjust settings like:
   - Window blur
   - Result spacing
   - Font size
4. Changes are saved automatically
