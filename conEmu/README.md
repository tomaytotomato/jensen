# Jensen Theme for ConEmu

Bring the Cyber Renaissance to ConEmu with this theme inspired by the iconic aesthetics of **Deus Ex: Human Revolution**. Featuring signature gold and copper tones set against dark backgrounds, Jensen transforms your Windows terminal into an augmented experience.

![Jensen Theme](../public/jensen_large.png)

**Creator:** tomaytotomato

## Features

- 🎨 **Cyber Renaissance Aesthetic**: Rich gold (`#EDA622`) and copper tones against deep carbon black backgrounds
- 🌙 **Dark Theme Optimized**: Easy on the eyes during extended terminal sessions
- 🎯 **Carefully Crafted Colors**: Optimized for readability
- 💻 **Windows Compatible**: Works with ConEmu on Windows

## Installation

### Method 1: Import XML

1. Open ConEmu
2. Open **Settings** (`Win+Alt+P` or Settings button)
3. Navigate to **Features** → **Colors**
4. Click **Import...** at the bottom
5. Select `themer-my-color-set-dark.xml` from this directory
6. Choose **Import scheme as...** and name it "Jensen"
7. Click **Save settings**

### Method 2: Manual Configuration

1. Open ConEmu Settings
2. Go to **Features** → **Colors**
3. Create a new palette named "Jensen"
4. Manually set each color from the values in the XML file
5. Save settings

## Additional Settings

For the best Jensen experience in ConEmu:

1. **Font**: Use a monospace font like Consolas or Fira Code
   - Settings → **General** → **Fonts**

2. **Transparency**: Add subtle transparency
   - Settings → **General** → **Appearance**
   - Set Alpha to 230-240

3. **Quake Style**: Enable dropdown console for an augmented feel
   - Settings → **General** → **Quake style**

## Tips

- **Multiple Consoles**: Jensen works great with ConEmu's multiple console tabs
- **Color Test**: Run `color` command to see all colors
- **Export Settings**: Export your ConEmu config to backup Jensen theme
- **Task Icons**: Customize task icons to match the gold theme
