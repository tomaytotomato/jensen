# Jensen Theme for BBEdit

Bring the Cyber Renaissance to BBEdit with this theme inspired by the iconic aesthetics of **Deus Ex: Human Revolution**. Featuring signature gold and copper tones set against dark backgrounds, Jensen transforms your editor into an augmented experience.

![Jensen Theme](../public/jensen_large.png)

**Creator:** tomaytotomato

## Features

- 🎨 **Cyber Renaissance Aesthetic**: Rich gold (`#EDA622`) and copper tones against deep carbon black backgrounds
- 🌙 **Dark Theme Optimized**: Easy on the eyes during extended coding sessions
- 🎯 **Carefully Crafted Colors**: Optimized for readability and visual hierarchy
- 💻 **macOS Native**: Designed specifically for BBEdit on macOS

## Installation

1. Copy `jensen.bbColorScheme` to BBEdit's Color Schemes directory:
   ```bash
   mkdir -p ~/Library/Application\ Support/BBEdit/Color\ Schemes/
   cp jensen.bbColorScheme ~/Library/Application\ Support/BBEdit/Color\ Schemes/
   ```
2. Open BBEdit
3. Go to **BBEdit** → **Preferences** (or press `Cmd+,`)
4. Navigate to **Appearance** → **Colors**
5. Select **Jensen** from the Color Scheme dropdown
6. Close Preferences - the theme is now applied!

## Alternative Method

You can also double-click the `jensen.bbColorScheme` file, and BBEdit will automatically import and apply it.

| **Eerie Black**   | `#141514`   | Main background, canvas                                  |

| **Imperial Red**  | `#ED333C`   | Error states, warnings, critical alerts                  |

## Tips

- **Font Recommendation**: BBEdit works great with SF Mono or Menlo (built into macOS)
- **Dark Mode**: Enable macOS Dark Mode for a fully immersive augmented experience
- **Sidebar**: The dark theme works beautifully with BBEdit's sidebar and project drawer
- **Export**: You can export your BBEdit settings to sync Jensen across multiple Macs

## Why BBEdit?

BBEdit is a powerful text and code editor for macOS with excellent syntax highlighting support. The Jensen theme takes full advantage of BBEdit's advanced color customization options.
