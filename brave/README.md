# Jensen Theme for Brave

Bring the Cyber Renaissance to Brave Browser with this theme inspired by the iconic aesthetics of **Deus Ex: Human Revolution**. Featuring signature gold and copper tones set against dark backgrounds, Jensen transforms your browser into an augmented experience.

![Jensen Theme](../public/jensen_large.png)

## Features

- 🎨 **Cyber Renaissance Aesthetic**: Rich gold (`#EDA622`) and copper tones against deep carbon black backgrounds
- 🌙 **Dark Theme Optimized**: Easy on the eyes during extended browsing sessions
- 🎯 **Carefully Crafted Colors**: Optimized for Brave's UI elements
- 💻 **Brave Compatible**: Works with Brave Browser on Windows, macOS, and Linux
- 🛡️ **Privacy-First**: Pairs perfectly with Brave's privacy features

## Installation

### Method 1: Load Unpacked Extension (Development Mode)

1. Download this directory or clone the repository
2. Open Brave and navigate to `brave://extensions/`
3. Enable **Developer mode** (toggle in the top-right corner)
4. Click **Load unpacked**
5. Select this `brave` directory
6. The Jensen theme will be applied automatically

### Method 2: Chrome Web Store Compatible

Since Brave is Chromium-based, you can also install Chrome themes:

1. Visit the Chrome Web Store (if Jensen is published there)
2. Search for "Jensen Theme"
3. Click **Add to Brave**
4. The theme will be applied instantly

- **Sync**: Brave themes sync across devices when signed in

## Troubleshooting

- **Theme Not Applying**: Make sure Developer mode is enabled and the extension is active
- **Updates**: If you update the theme files, reload the extension on `brave://extensions/`
- **Conflicts**: Disable other themes before applying Jensen
- **Reset**: Go to `brave://settings/appearance` to reset or change themes
